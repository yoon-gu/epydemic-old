# epydemic

## Installation
http://thenilblog.blogspot.kr/2015/01/install-pyqt-on-yosemite.html

### Install Qt
1. brew install qt

### Install SIP
1. download SIP
1. python configure.py
1. make
1. sudo make install

### Install PyQt4
1. download PyQt4
1. python configure.py
1. make
1. sudo make install

### Install pyqtgraph
1. Download pyqtgraph
1. python setup.py install

### Install pyopengl
1. sudo pip install pyopengl
